# GA Snake

Example of game snake with Genetic Algorithm on server

# Getting started

Copy project

```
git clone https://github.com/AlecBold/GASnake.git
```

# Prerequisites

python 3.6.8 or higher

node v8.10.0

# Run
```
python3 serve.py
```
or
```
python serve.py
```
